# 移通教务系统自动评教助手

自动处理教务系统评教表单的脚本工具。

## 功能特性
- **全自动流程**：支持自动选择未评条目、填充评分、滚动到底部及自动翻页。
- **模拟人工**：全流程添加 1 秒操作等待，降低被检测风险。
- **智能定位**：填写完成后自动滚动页面，方便人工快速检查。

## 注意事项 ⚠️
- **安全提交**：由于系统可能存在机器人行为检测，脚本填充完成后**建议手动点击提交**。
- **环境限制**：目前仅适配[**移通学院内网教务系统**](http://10.252.6.31/jwglxt/xspjgl)环境。
- **手动适配**：外网地址经常变动，若需在其他域名下使用，请自行修改脚本头部的 `@match` 参数。
