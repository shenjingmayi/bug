# 米游社改造计划 - 油猴脚本

这是一个功能强大的米游社（miyoushe.com）页面定制油猴脚本，旨在通过深度优化 UI 和移除冗余元素，为您提供一个更加纯净、高效的浏览体验。

## 🚀 主要功能

### 1. 顶部导航栏优化
- **Logo 移除**：精简顶部空间，移除官方 Logo。
- **标题精简**：自动去除标题中的“米游社”字样及多余空格。
- **导航管理**：内置可视化管理面板，可自由选择导航项的**显示**、**收纳（至更多）**或**彻底隐藏**。
- **通知清理**：可选隐藏右上角的关注和通知图标。

### 2. 页面布局自定义
- **右侧边栏移除**：一键移除右侧复杂的侧边栏区域。
- **宽度自由调节**：支持手动设置页面内容区域（`.mhy-layout`）和主信息区域（`.mhy-layout__main`）的宽度（单位：px）。
- **多布局同步**：自动识别并同步设置页面中所有的布局容器，确保体验一致。

### 3. 内容过滤与清理
- **创作者/作者信息**：移除侧边栏的创作者操作区及文章页的作者信息。
- **广告与推广**：移除侧边广告轮播、侧边栏推广位、页脚广告卡片等。
- **游戏工具区**：移除侧边栏的游戏工具入口。
- **标题过滤**：支持通过关键字过滤侧边栏板块。

### 4. 细节优化
- **间距压缩**：移除文章段落间多余的行间距。
- **组件移除**：可选移除左侧操作栏、底部公司信息、小火箭回到顶部组件等。

## 🛠️ 安装方法

1. 首先确保您的浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 插件。
2. 点击新建脚本，将 `miyoushe.js` 中的内容全部复制并粘贴进去。
3. 保存脚本并刷新米游社页面即可。

## ⚙️ 使用说明

- **打开面板**：点击页面左下角的 **⚙️ (齿轮)** 图标即可唤起配置面板。
- **即时生效**：大部分选项在点击后会立即应用，无需刷新页面。
- **宽度设置**：勾选“布局宽度设置 (px)”左侧的复选框后，下方的自定义宽度才会生效。
- **导航管理**：点击面板中的“导航收纳管理”按钮，可进入可视化弹窗进行详细配置。
